一変数の場合[編集]

a, b は実数の定数とするとき、

a x + b = 0 {\displaystyle ax+b=0} または a x = − b {\displaystyle ax=-b}

なる形をとる。後者の形の場合は、a ≠ 0 ならば（a−1 = 1⁄a が存在するから）一意的に解けて x = b⁄a がその解である。a = 0 のとき、b ≠ 0 ならば不能、b = 0 ならば不定である。

二変数の場合[編集]

詳細は「一次函数」を参照

一般形は

a x + b y = -c {\displaystyle ax+by+c=0}

で、これは {(x, y) | ax + by + c = 0} なる集合、つまり平面上の直線を表すと考えられる。直線が座標軸と平行でない場合、

y = a x + b {\displaystyle y=mx+b}

なる形で扱うことができる。これはふつう、x を自由変数とし y を x の従属変数とみるとき、一次函数と呼ぶ。
