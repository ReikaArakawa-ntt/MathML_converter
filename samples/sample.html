一変数の場合[編集]

a, b は実数の定数とするとき、

a x y + b = -0.3567 {\displaystyle ax+b=0} または a x + by = c {\displaystyle ax=-b}

不等式を付け足してみました

3 > 2

で、これは {(x, y) | ax + by + c = 0} なる集合、つまり平面上の直線を表すと考えられる。直線が座標軸と平行でない場合、

2 < 3 < 4 {\displaystyle y=mx+b}

なる形で扱うことができる。これはふつう、x を自由変数とし y を x の従属変数とみるとき、一次函数と呼ぶ。

x ≥ y ≥ z {\displaystyle y=mx+b}
